## 🚀 Deployment Instructions

### 1. Upload Theme
```bash
cd basic_shopify_theme
shopify theme push
```

### 2. Configure in Shopify Admin
1. **Homepage:** Customize sections in Theme Editor
2. **Products:** Assign `product.styledrop` template
3. **Collections:** Assign `collection` or `collection.brand` templates
4. **Brands Page:** Create page, assign `page.brands` template

### 3. Add Content
- Upload product images
- Add product tags (`fast-drop`, `new`, `best-seller`)
- Set product vendors for brand filtering
- Configure menus (`main-menu`, `footer-shop`, `footer-support`, `footer-quick`)
